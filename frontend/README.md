# Cleaning Reports Frontend

Vite + React frontend with routing and API client.

## Quick start

```bash
cd cleaning-reports/frontend
npm install
npm run dev
```

Visit http://localhost:3000 (default port, proxied to http://localhost:4000/api).

## Pages

- `/login` — Login page
- `/` — Dashboard
- `/buildings` — Building management
- `/inspections` — Inspection reports
- `/tasks` — Cleaning tasks
- `/claims` — Client claims
- `/reports` — Reports and exports

See `src/api.js` for all API methods.
